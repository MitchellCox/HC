The Layers panel is ideal for separating elements in a diagram and viewing or hiding them. By default, the editor has no layers. To hide, show, lock, or add interactivity to areas, objects must be placed on a layer. In advanced diagrams like mockups and wireframes, hotspots are used to create interaction; for example, clicking a button can hide or show the specified layers.

Layers

A layer is an alternative view on a page. It can be shown or hidden, depending on the action assigned to the hotspot connected to it. Layers allow you to demonstrate user interaction before writing any code. To access layers, select “Layers” from the Right Menu.

Layers Panel Overview:

  • Layers List: Default location of all layers associated with the current page. If you are zoomed in or working on a large diagram, click a layer in the list to view that layer's location on the page.
  • “+Layer”: Create a new layer.
  • “Page New Page”: Default location of all objects added to the editor.
  • Eye Icon: Temporarily hide or show the layer. This is also used to sync the visibility of the published diagram.
  • Lock Icon: Lock a layer so it can no longer be edited. You can unlock by clicking the lock icon in the Options Bar or the Layers Panel.
  • Pen Icon: Rename, delete, or duplicate the layer.
  • Sync Visibility: Set the initial visibility state of the published document to match the current visibility state. To do so, select the eye icons for the layers that should appear by default, then click the Sync Visibility button.

Create a layer:

  1. Click “Layers” from the Options Bar.
  2. Choose the “+Layer” button.
  3. Type a title for the layer. If you want to change the title, right-click on the current title and select “Rename”.
  4. To add shapes to a layer, right-click on a shape and choose “Move to Layer”, or double-click on a layer and drag shapes onto the canvas.

Edit a layer:

From the Layers panel, double-click on a layer to edit it. A blue bar will appear on the left side of the layer when it is selected. All items not in that layer will be grayed out on the canvas.

If you click on a layer on the canvas that you aren't currently editing, you'll see a bounding box around all objects on that layer (see image below). If you double-click that layer, editing will be enabled.

Hide/show a layer:

To view a layer, click its eye icon. Remember that if you use the quick preview hotkey (Ctrl+Shift or Cmd+Shift) and click on a hotspot, the eye will be temporarily unsynced.

Lock a layer:

Locking a layer will freeze all objects on the layer. You'll no longer be able to edit the layer. To lock multiple layers at a time, hold Ctrl (Cmd on Mac) and click each layer. The layer must be unselected in order to lock it.

Duplicate a layer:

Duplicating a layer will create a copy in the same place as the original layer. You can choose the page on which a duplicate layer will appear.

To duplicate a layer, right-click or choose the pen icon and select “Duplicate”. Then select the destination page or create a new one.

Reorder layers:

Click and drag to reorder layers.

Move to layers:

  1. Select object(s) from the page or the layer you're currently editing.
  2. Right-click and select 'Move to Layer'.
  3. Choose the destination layer or create a new layer, then click 'OK'.

Copy to layers:

  1. Select object(s) from the page or the layer you're currently editing.
  2. Right-click and select “Copy to Layer”.
  3. Choose the destination layer or create a new layer.

Sync Visibility:

This feature establishes the default view for users who access a published webpage version of the diagram.

  1. Click the eye icon of each layer until the page looks the way it should appear when it's published.
  2. Click the Sync Visibility icon. When the icon is grayed out, syncing is complete.
  3. When you update your diagram, it will update on the webpage as well.

Hotspots

A "hotspot" is a designated area of the canvas that performs an action when selected in Present mode. Hotspots are ideal for UI Mockups and demonstrations.

To create a hotspot:

  1. Drag and drop the green hotspot shape from the standard shape library onto the selected object.You can also right click on a shape to add a hotspot.
  2. Click on the drop down arrows next to "Choose an Action," and select one of the six actions.
    • Toggle layers: When you click the selected shape in present mode, it will toggle between hiding and showing everything in the selected layer.
    • Show layers: When you click the selected shape in present mode, it will show all objects in the selected layer.
    • Hide layers: When you click the selected shape in present mode, it will hide all objects in the selected layer.
    • Link to page: When you click the selected shape in present mode, it will take you to the selected page.
    • External link: When you click the selected shape in present mode, it will take you to the web URL you entered.
    • Link to email: When you click the selected shape in present mode, it will take you to the email you entered.
  3. If you've chosen a "link" option, enter the required information and click "Done." If you've chosen an action that works with layers, you'll need to select a layer on which the hotspot will act.

If you have not created a layer yet...

  1. Type a title for the layer into the "Create New Layer" field.
  2. Click the [+] sign to create the layer.
  3. Select the layer from the list, and click "Done."
