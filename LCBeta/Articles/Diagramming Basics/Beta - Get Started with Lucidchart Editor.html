The editor is where the magic happens, and it holds the key components for bringing your diagram to life -- the Canvas, Shape Toolbox, Options Bar, Page Drawer, Right Menu, File Menu, and Doc Icon. After reading this tutorial, you’ll be set to start navigating the editor. The editor is compatible with any web browser.

Canvas: The white space where you create.

  • Resize to accommodate large and small diagrams.
  • Format dimensions, orientation, and margins of each page.
  • Toggle the grid on and off.
  • Place your cursor over the canvas to drag it left or right.

Shape Toolbox: Located to the left of the canvas, the Toolbox stores all your shape libraries.

  • View more libraries by clicking “Shapes” to open the Shape Manager and add new libraries to the Toolbox.
  • Search Lucidchart, Iconfinder, and Google Images.
  • Import images, SVG files, and Microsoft Visio stencils.
  • Create custom libraries.
  • Search for specific shapes.
  • Add your favorites to the “My Saved Shapes” library.
  • Resize the Toolbox by clicking and dragging the handle on the right side.

Options Bar: Located along the top of the canvas, the Options Bar allows you to format shapes, text, and lines, as well as access advanced features such as adding actions to and locking shapes. Call-outs appear when you click on some icons, and you can pin those call-outs so that they remain open in the editor. A "More" icon may appear at the end of the Options Bar to hold icons that don't fit.

 Undo or redo.

 Choose text font and size.

 Bold, Italics, Underline Icons: Bold, italicize, or underline text.

 Alignment Icon: Adjust text alignment.

 Text Options Icon: Add subscript, superscripts, and strikethroughs, change line height, choose a list type (bullet, numbers, etc.), adjust indent size and padding, and rotate text.

 Color Icon: Change the color and opacity of a shape’s fill, border, text, or line. Select an image as a shape’s background by selecting “More” under the “Fill” tab.

 Shape Options Icon:Adjust border thickness, style (solid, dotted, or dashed), and roundness of corners and shape opacity. Click the "Shadow" dropdown menu to turn shape shadow on or off. When the shadow is on, change the distance, blur, color, and angle of the shadow.

 Position Icon: Change the shape’s width, height, rotation, or position on page and flip the shape horizontally or vertically.

 Line Style Icon: Change the shape of the line (curved, elbow, or direct). Create a double line.

 Line Icon: Change the line thickness, line style (solid, dashed, or dotted), arrow type, and roundness of corners.

 Arrow Icon: Change the direction of the arrow.

 Actions Icon: Add an action to a shape, such as linking it to another page.

 Magnetize Icon: If the magnet is on (blue), any content inside a container will move with the container.

 Lock Icon: If a shape is locked (blue), the shape can no longer be edited.

 Shape Data Icon: Add data to an object, such as an IP address.

 Advanced Shapes Icon: Open and close the Advanced Shapes Bar.

Page Drawer: Located at the bottom of the editor, the Page Drawer manages all your pages within a document. You can manage pages by opening the Page Drawer or by using the Page Tab.

  • Create a new page within a document.
  • Rename pages.
  • Navigate between pages.
  • Create a copy of a page.
  • Adjust order of pages.
  • Delete pages.
  • Create and apply a Master Page.

Right Menu: Located to the right of the canvas, the Right Menu is your go-to for page settings, sharing, and comments.

  • Customize page, margin, grid, and line settings.
  • Make comments on shapes or on the entire document.
  • Create presentation slides.
  • See revision history.
  • Create and manage layers. 
  • Add a theme.
  • Chat with collaborators. 

File Menu: Located along the top of the editor, the File Menu contains the File, Edit, View, Insert, Arrange, Share and Help dropdown menus.

  • Access all your documents.
  • Create a new document.
  • Duplicate a document.
  • Download a document as PDF, PNG, JPEG, Visio, SVG.
  • Import a document.
  • Share, embed, publish or print a document.
  • Insert a new page, comment, image, presentation slide, etc.
  • Adjust the view of your document.
  • Arrange objects in your document.
  • Access helpful resources.

Doc Icon: Located in the top left corner of the editor, the Doc Icon takes you back to the documents list.
