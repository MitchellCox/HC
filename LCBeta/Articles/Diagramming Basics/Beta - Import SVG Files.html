SVG files can be downloaded from the Internet or created in programs such as Adobe Illustrator, Inkscape, and SVG-edit.

To import directly to the canvas:

  1. Click the image icon at the top of the Shape Toolbox and select “SVG” OR select the Insert dropdown menu and select "SVG."
  2. Select an SVG file to upload.
  3. The SVG will not be saved unless you drag it into “My Saved Shapes” or create a custom library.

To import in the Shapes Manager:

  1. Click “Shapes” on the Shape Toolbox.
  2. Select “Import” > “SVG”.
  3. Select whether you would like the SVG to import to “My Saved Shapes” or to a new library. Select an SVG file and click “Import”.
