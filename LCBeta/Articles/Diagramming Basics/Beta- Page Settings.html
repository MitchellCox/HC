You can customize your page, margin, guides, and line settings by selecting “Settings” from the Right Menu or selecting "Page Settings" from the File dropdown menu.

Page:

  1. Change the page size by choosing from the options in the dropdown menu. Select “Custom” to input your own dimensions.
  2. Choose portrait or landscape orientation.
  3. Select your measurement unit.
  4. Choose whether you would like page auto resize on or off by turning page tiling on or off.
  5. Select a background color to fill the entire page.
  6. Use the content scale to adjust how large or small your diagram is in relation to the page size.
  7. Choose “Fit” to have everything on the diagram fill the page.

Margin:

  1. Select whether you would like the margin on or off. If the margin is on, it will be displayed as a blue border around the page.
  2. Customize the margin size.

Guides:

Guides are helpful for creating diagrams such as mockups and wireframes in order to align objects. To create a guide, right click on the canvas and select “Create Horizontal Guide” or “Create Vertical Guide”. To delete a guide, right click on the canvas and select “Delete Horizontal Guide” or “Delete Vertical Guide”.

  1. Select whether you would like guides on or off.
  2. Choose if you would like guide snap on or off.

Lines:

  1. Choose if you would like to be able to drag a line from a shape automatically by using the “On” / “Off” toggle.
  2. Choose if you would like line jumps on or off. Enabling line jumps create a little “bump” when lines cross, while disabling line jumps creates intersecting lines.
