Lucidchart allows you to import diagrams from MS Visio (.vdx), Omnigraffle (.graffle, /graffle.zip), and Gliffy (.gxml, .gliffy).

Import from the Documents page:

  1. Select “Import”.
  2. Click “Import Visio”, “Import Gliffy”, or “Import OmniGraffle”.
  3. Click “Choose Files” to select the files you would like to import.
  4. Imported files will automatically appear in your Documents list.

Import from within the editor:

  1. Click the File dropdown menu and select “Import Diagram."
  2. Click “Visio," “Omnigraffle," or “Gliffy."
  3. Click “Choose Files” to select the files you would like to import.
  4. Imported files will be saved to the Documents page and will not be found in the current document.
