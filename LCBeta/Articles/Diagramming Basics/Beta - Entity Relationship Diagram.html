Lucidchart allows you to create entity relationship diagrams manually using the ERD shape library or automatically using the import feature.

Automated ERD Creation

The automated ERD feature enables you to automatically generate all of your database tables and schemas directly into the toolbox. When related tables are dragged to the canvas, the relationship lines will also be drawn automatically. Updated database files can be imported at any time, and the fields will be be updated in the toolbox and on the canvas.

  1. Click “Shapes” on the Shapes Toolbar.
  2. Drag and drop or pin the ERD shape library to the Shapes Toolbox.
  3. Click “Import” in the ERD shape library.
  4. Choose the Database Management System you use.
  5. Copy the provided query and paste it into the DBMS. Run the query.
  6. Export the result from your DBMS as a .tsv or .csv and upload it to this page, OR copy the result and paste it into the open text area.
  7. The text will be automatically imported and converted into an organized list of all databases, tables and schemas. Each collapsable section represents a database, and the names inside correspond with the database tables. An additional subsection would indicate a schema.

Manual ERD Creation

  1. Click “Shapes” on the Shapes Toolbar.
  2. Drag and drop or pin the ERD shape library to the Shapes Toolbox.
  3. Drag and drop an entity object onto the canvas.
  4. The Advanced Shapes Bar will appear along the top of the canvas as soon as you drop an object and can be used to format your ERD.

You can open or close the Advanced Shapes Bar with the tool icon in the Options Bar.

The Advanced Shapes Bar allows you to:

  • Add or remove fields.
  • Insert a field above or below another field.
  • Delete a specific field.
  • Shade the header.
  • Shade alternate rows.
  • Export to SQL.
