When you share a document, you select the permission level of the collaborator. Some of the actions available to the collaborator in “View Only” mode include:

  • Select the File menu to access other documents, create a new document, download the document, print the document, etc.
  • Adjust account settings and user preferences.
  • Enter and exit full-screen mode.
  • Switch between pages using the Page Drawer.
  • Zoom in or out.
  • Access the Help Center.
