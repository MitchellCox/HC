Easily customize your shapes by using the Options Bar to change the shape’s size, position, rotation, border, or shadow, as well as to add actions, lock, or magnetize a shape.

Change the shape size:

  1. Select the shape.
  2. Grab a corner of the shape and drag to resize it.

OR

  1. Click the Position icon in the Options Bar.
  2. Adjust the width and height of the shape.

Move the shape:

  1. Select the shape.
  2. Drag and drop the shape to its new location on the canvas.

OR

  1. Select the shape.
  2. Click the Position icon in the Options Bar.
  3. Adjust the X and Y Positions to move the shape horizontally or vertically on the page.

Rotate the shape:

  1. Select the shape.
  2. Click and drag the curved arrow icon outside the top left corner of the shape to rotate it.

OR

  1. Select the shape.
  2. Click the Position icon in the Options Bar.
  3. Adjust the shape’s rotation.

Change the shape border:

  1. Select the shape.
  2. Click the Shape Options icon in the Options Bar, and choose a thickness, style, radius, and more.

Change the shape color:

  1. Select the shape.
  2. Click the Color icon in the Options Bar.
  3. Choose “Fill”, “Border”, or “Text” and select a color. “Fill” changes the whole background color. “Border” changes the border color. “Text” changes the font color.

Change the shape shadow:

  1. Select the shape.
  2. Click the Shape Options Icon in the Options Bar, and then click the Shadow dropdown.
  3. Turn the shadow on or off.
  4. If the shadow is on, adjust the shadow’s color, distance, blur, and angle.

Lock the shape:

  1. Select the shape.
  2. Click the Lock icon in the Options Bar. When the lock is blue, the selected shape can no longer be edited.

Add an action to the shape:

  1. Select the shape.
  2. Click the Action icon in the Options Bar.
  3. Choose an action to add.

Magnetize the shape:

  1. Select the shape - remember that you can only magnetize certain shapes, such as containers.
  2. Click the Magnetize icon in the Options Bar. When the magnet is blue, any content inside the selected object will move with the object.
