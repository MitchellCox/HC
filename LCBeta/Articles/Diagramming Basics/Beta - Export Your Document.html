You can export your Lucidchart diagrams into PDF, PNG, JPEG, SVG, and VDX file formats by following these steps:

  1. Open the diagram you want to export.
  2. Open the File dropdown menu and select “Export & Download”.
  3. Choose the file type in which you would like to export the diagram, as well as the content you would like to export. If you are exporting as an image, select the image quality.
  4. Click “Download” and the file will download to your computer in the desired format.
