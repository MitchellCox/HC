To import directly to the canvas:

  1. Click the image icon at the top of the Shape Toolbox and select “Image”

  2. Select an image file from your computer to import.
  3. The imported image will appear on the canvas.
  4. Your image will not be saved unless you drag it into “My Saved Shapes” or create a custom library.

To import in the Shapes Library:

  1. Click “Shapes” on the Shape Toolbox.
  2. Select “Import” > “Image”.

  3. Select whether you would like the image to import to “My Saved Shapes” or to a new library.
  4. Select an image file and click “Import”.

To import from Google Images:

  1. Click the search icon at the top of the Shape Toolbox.
  2. Select “Google."
  3. Search for an image.
  4. Drag and drop an image onto the canvas.

To import from Iconfinder:

  1. Click the search icon at the top of the Shape Toolbox.
  2. Select “Iconfinder."
  3. Search for an icon.
  4. Drag and drop the icon onto the canvas.

You can edit your imported image by right clicking and selecting "Edit Mask". Doing so allows you to crop your image.
