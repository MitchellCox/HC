The Shape Manager allows you to create your own custom shape libraries. You can share these libraries with others and access libraries shared with you.

**Note: If a library is shared with you, you can’t drag additional shapes into the shared library.

To create a new custom group:

  1. Open the Shape Manager by clicking “Shapes” in the Toolbox.
  2. Select “+ Custom Group”. You can also right click anywhere on the canvas to add a custom shape library.
  3. Drag and drop shapes from any existing library into your custom library.
  4. To name the new library, click the current title and type in the new name, or click the settings icon in the top right corner of the library and select “Rename”.
  5. To share the library, click the settings icon in the top right corner of the library and select “Share”.
  6. To delete the library, click the settings icon in the top right corner of the library and select “Delete”.
