When you share a document, you select the permission level of the collaborator. Some of the actions available to the collaborator in “Comment Only” mode include:

  • Select “Comment” from the Right Menu to comment on a shape or on the entire document, view all comments, and reply to comments. For a more comprehensive look at commenting, visit our commenting tutorial.
  • Select the File menu to access other documents, create a new document,  download the document, share the document, or print the document.
  • Adjust account settings and user preferences.
  • Enter and exit full-screen mode.
  • Switch between pages using the Page Drawer.
  • Zoom in or out.
  • Access the Help Center.
