Adding a password to your published documents is a useful way to enhance security. Others trying to access a password protected document will not only need the specific URL, but the password as well.

To publish a document with a password, use the following steps:

  1. From the File menu, select “Share” and then “Publish.”
  2. Under the “Publish” tab, select “Full document (URL).”

  3. Select “Require Password, then enter desired password.

  4. Press the blue “Generate Link” button.
  5. Press the blue hyperlink button to the left of “Full document (URL).”
  6. Copy and paste the link.
  7. Users who navigate to the generated URL will be required to enter the password you set before viewing the document.

Note: This feature is available for Team Accounts.
