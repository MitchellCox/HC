It's easy to create a new document in Lucidpress. You can start from scratch or select a template from our collection.

Creating a New Document

  1. If you are not already in your Documents List, select the 'Documents' link in the upper right corner.

  2. Click the green '+ Document' button. 

  3. Decide whether you're making a print or digital document (formatted for easy viewing on a computer, tablet, or smartphone).

  4. Check to see if we have examples for the type of document you are trying to create. If we do, select the category name to see available templates.

  5. You can hover over a template and click the blue 'Preview' button to swipe through the pages to evaluate each template in greater detail. 

  6. Click your desired template to begin editing.

  7. If you wish to create a blank document, click the blue 'Create New' button.
