Looking for help importing your InDesign files? This tutorial will show you how.

How to Import an .indd File in Lucidpress
To import your .indd file, you’ll need to convert it to Adobe’s .idml file format.

If you have Adobe InDesign:

  1. Open the indd file and click File > Export from the File menu. 
    • Simply select the .idml format for output and save.

If you don’t have Adobe InDesign:

  • Contact the designer of the document and ask for the file in .idml format.
  • Use the free online tool iXentric SaveBack to convert your .indd files to the .idml format. Please note that iXentric SaveBack does not yet support InDesign CC 2015 and Lucidpress and Lucid Software are not affiliated with this service in any way. You may be able to get satisfactory results from other online .indd > .idml conversion tools.
How to Fix Missing Images in your Imported InDesign Document
  • InDesign documents don’t store image files by default. If an imported InDesign document doesn’t have stored images, then they must be manually uploaded. To upload them, select all images in your folder and we’ll automatically add them to the document.
How to Fix Missing Fonts in your Imported InDesign Document
  • InDesign documents don’t store font files by default. To access your fonts, upload them to Lucidpress.
 
Review your Document
  • Click on a thumbnail to see the pages.
 
 

  • If an image is missing, you’ll see a red triangle warning and a placeholder image. If a font is missing, you’ll see the same red triangle and highlighted text. Click to open the upload dialog again and identify the missing content.
