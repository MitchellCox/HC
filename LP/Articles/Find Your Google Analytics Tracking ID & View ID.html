For Google Analytics to work with Lucidpress, you will need to provide the View ID associated with the property of the Tracking ID you are using to track Lucidpress. This tutorial assumes you have already created a Google Analytics account, and a property for Lucidpress’ analytics. A tutorial on creating a Google Analytics Account can be found here, and another on creating a property in Google Analytics here.

  1. Go to the Google Analytics sign in page.


  2. Sign in to your account or click ‘Access Analytics’ if you are already signed in.
  3. Click on the ‘Admin’ tab.
    Note:     If you have multiple accounts, select the account in which you want to store the Lucidpress data.


  4. Select the Property you will be using to track your Lucidpress analytics using the drop-down menu below ‘Property’.

  5. If you haven't copied and pasted your Google Analytics Tracking ID (begins with UA-) into the Lucidpress Analytics set up page, do so now using the number displayed under the 'Property' section.






  6. Select ‘View Settings’ beneath ‘View’.


  7. The View ID for the selected property is listed first under ‘Basic Settings’. Copy View ID to the clipboard and paste it into the required field when setting up your Lucidpress Google Analytics.


  8. For additional setup instructions, see our tutorial covering how to set up Lucidpress published document analytics.
