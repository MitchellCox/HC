If you're looking to print your document professionally through a print shop, you probably want a very specific set of colors to show in the printed copy--this requires translation from screen color (RGB) to print color (CMYK).

In order to correctly translate the RGB color you used in Lucidpress into CMYK color on the printed page, you will need to apply a special set of instructions to your PDF or JPG file, called a "Color Profile."

1) While in the Editor, click File > Download as in the Menu bar, or use the keyboard shortcuts.


2) Select PDF, PNG, or JPEG, and click on the drop down for "Color Profiles."



Note: We have RGB and CMYK color profiles available, in case you want to slightly change the default setting.

3) After selecting a Color Profile, press "Download," and your file will download with those instructions embedded into the structure of the file. When you take it to a print shop, they will interpret the file and use the color profile we saved with the download.



Some at-home printers market a variety of features as "CMYK" support, but they are not all the same, and do not follow a consistent set of instructions. If you have any questions about how to print in CMYK, contact your printer's manufacturer (HP, Brother, etc), explore their website, or send your file to a print shop for a single printing.
