In order to print with Blurb, some special steps need to be taken to ensure print compatibility.

Remove Searchable Text
  1. Click this link, and then download the document you wish to use with Blurb. This link will tell your browser to temporarily remove searchable text, which often causes errors when printing. The link will only work once, and will need to be used again for each document downloaded.

  2. Remove Searchable Text Link

 

Convert to PDF/A
  1. Using a third party PDF editing software, like Adobe Acrobat Pro, convert your PDF to PDF/A. This format is most compatible with Blurb, and will help ensure an accurate and easy print job. 



  2. Upload to Blurb as a PDF
