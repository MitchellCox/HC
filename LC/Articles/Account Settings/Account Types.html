Lucidchart is an online diagramming tool used to visualize processes, models, and other diagrams. Users can create a new document from one of our helpful templates, or create custom diagrams from scratch. Lucidchart lets you share diagrams with any user on any operating system, edit together, and sync everything instantly.

This article is a high-level overview of Lucidchart's various account types. For more detailed information, visit our pricing page.

Free Account
  • 5 document limit
  • 60 object limit per document (All lines, shapes, and images count as individual objects towards the 60 object limit)
  • 25 MB storage
Basic/Personal Account
  • Unlimited amount of documents
  • Unlimited amount of objects per document
  • 100 MB of storage
  • Basic shape libraries
  • Access to layers
Professional Account
  • 1 GB of storage
  • Presentation mode
  • Visio export
  • VDX, Gliffy, and Omnigraffle import
  • VSDX/SVG import
  • Advanced shape libraries (iOS mockups, UI mockups, Mind Mapping)
  • Document password protection
  • Back up and Restore
Team Account
  • 5 GB of team storage
  • Shared templates and images
  • Confluence integration
  • Google Apps for domain
  • Basic sharing and publishing control
  • Add up to 100 users
Enterprise Account
  • 5 GB of storage per user
  • Dedicated Customer Success Associate
  • Enterprise-wide visibility for all users on account
  • Document retention features
  • Advanced sharing control
  • Advanced publishing control
  • More Integrations (SAML, Jira, Gemini, Box)
  • Data Linking
