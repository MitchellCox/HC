The Layers panel is ideal for separating elements in a diagram and viewing or hiding them. This tutorial will show you how to create and manage layers.

Overview of the Layers panel

By default, the editor has no layers. To hide, show, lock, or add interactivity to certain areas, you must place objects on a layer. In advanced diagrams like mockups and wireframes, hotspots are used to create interaction. For example, clicking a button can hide or show the specified layers. Read our Help Center article about adding interaction to learn more.

Components of the Layers panel include:

  • Page: This is the default location of all objects added to the editor.
  • Layers list: This is the default location of all layers associated with the current page. If you are zoomed in or working on a large diagram, click a layer in the list to review that layer’s location on the page.
  • Hide/Show toggle: Click the eye icon to the left of the layer name to temporarily hide or show the layer. You can also use this feature to sync the visibility of the published diagram.
  • Reorder handle: Rearrange the order of layers using the six-dot icons to the right of the layer name.
  • Action buttons: These buttons are found below the layers list. Click a layer to select it, and then lock, rename, delete, or duplicate the layer.
  • Sync Visibility button: Set the initial visibility state of the published document to match the current visibility state. To do so, select the eye icons for the layers that should appear by default, and then click the Sync Visibility button.
  • Present mode: Place the diagram into a presentation mode with this button. It’s ideal for demonstrating interaction applied to the diagram, as well as hiding distracting toolbars.
  • + Layer: Create a new layer by clicking this button.

Create a layer

Click the “+ Layer” button to add a new layer.

Note: You can also create a new layer when a layer is moved or copied.

Edit a layer

From the Layers panel, double-click on a layer to edit it. The layer title will be highlighted, and you’ll also see a breadcrumb at the top of the editor that indicates which layer you’re in.

If you click on a layer on the canvas that you aren’t currently editing, you’ll see a bounding box around all objects on that layer (see image below). If you double-click that layer, you will enable editing abilities.

Hide/Show a layer

To view a layer, click its eye icon. Remember that, if you use the quick preview hotkey (Ctrl+Shift or Cmd+Shift) and click on a hotspot, the eye will be temporarily unsynced.

Lock a layer

To lock a layer, click the lock icon under the layers list. Locking a layer will freeze all objects on the layer. You’ll no longer be able to edit the layer.

To lock multiple layers at a time, hold Ctrl (Cmd on Mac) and click each layer. You cannot select multiple layers while you are editing a single layer.

Duplicate a layer

Duplicating a layer will create a copy in the same place as the original layer. You can choose the page on which a duplicate layer will appear.

To duplicate a layer, select it and click the duplicate icon. Then select the destination page or create a new one.

Reorder layers

Click on the six-dot icons on the right side of the layer list to drag the layers around.

Move and Copy objects to a layer

To move objects to a layer:

  1. Select object(s) from the page or the layer you’re currently editing.
  2. Right-click and select “Move to Layer.” If you want to move an object that’s currently placed in a layer, you need to right-click from that layer for the “Move to Layer” option to appear.
  3. Choose the destination layer or create a new layer, and then click “OK.” The objects will move to the destination layer.

      

To copy objects to a layer:

  1. Select object(s) from the page or the layer you’re currently editing.
  2. Right-click and select “Copy to Layer.” If you want to copy an object that’s currently placed in a layer, you need to right-click from that layer for the “Copy to Layer” option to appear.
  3. Choose the destination layer or create a new layer, and then click “OK.” The objects will be copied to the destination layer.

      
Use Sync Visibility

This feature establishes the default view for users who access a published webpage version of the diagram.

  1. Click the eye icon of each layer until the page looks the way it should appear when it’s published.
  2. Click the Sync Visibility icon. When the icon is grayed out, syncing is complete.

Note: Demo mode will show the same view as the synced version. However, actions taken in the demo mode will not affect the appearance of your document when you return to the editor.
