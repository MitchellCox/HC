Adding, editing, resizing, moving, and rotating text boxes

 

  1. Add a text box by clicking the appropriate icon in the Content bar and dragging it to the canvas.

  2. Once you have added a text box, you can begin typing immediately or copy and paste content to the box. If you have exited out of editing the text box, you can re-enter to edit the text by double-clicking.

  1. If your content overflows a text box, you will see a red + symbol at the bottom of the box. 

  2. Change the size of the text box by dragging the size handles on any corner.

  3. Click the canvas to exit text-editing, then click and drag the text box to the desired location.

  4. Alternatively, you can click the text box and use your arrow keys to nudge, or Shift +arrow key to make a leap.

  5. Rotate the text box by grabbing the rotate handle at the top of the box and dragging.

Linking multiple standard text boxes

  1. To link multiple text boxes, click the light blue box on the lower-right corner of the text box and drag the blue line to the upper-left corner of the next text box.

Basic text formatting

  1. Select the text you'd like to format by double-clicking the text box to enter edit mode. To select a portion of the text, click and drag to highlight the desired portion. 

  2. Change the font using the drop-down menu in the Text tab.

  3. Change text styling with the buttons in the Text tab.

  4. Change text size with the arrows in the Text tab.

  5. Change the text color by selecting a color from the Text tab. You can also enter a hexadecimal color code or an opacity percentage in this panel; press Enter to apply.

  Advanced text formatting

  1. Change text alignment using the 'Alignment' section of the Text tab.

  2. Add bullets to your text by using the 'Bullets & Lists' drop-down menu in the Text tab.

  3. Line spacing can also be altered from the Text tab under 'Spacing'. Enter a value and press Enter to apply.

  4. The Text tab allows you to modify the character style, outline, and highlight.

 Adding and uploading new fonts

 

  1. Select a text box.

  2. From the font selection menu in the Text tab of the Context pane, select 'Manage fonts'. This will open the Font Manager.

  3. In the Font Manager, you can view available fonts by category, preview individual fonts by selecting them, and add fonts to your in-editor list by turning them to 'On'. You can remove fonts from that list by turning fonts to 'Off'. Fonts that are turned on will be viewable on your 'Enabled fonts' list. Click 'Save' to save your changes.

  4. To add custom fonts, click 'Upload' and select the file (.ttf, .ttc, or .otf), then click 'Upload' to add the font. Then click 'Save changes' or simply drag the file to the editor.

 Wrapping text around objects (shapes, images, videos, and text boxes)

  1. Select the object that you would like to wrap text around (shape, image, gallery, video, or text box).

  2. Once the object is selected, go to the right pane and the Layout tab. Scroll down to Text Wrapping as seen in the above screen shot.

  3. To force the text wrap closer to the boundary, select 'Wrap tight to object boundary.'

  4. You can specify wrap preferences with the wrap format buttons.

  5. Right click on the object on the page and bring the object to the front. This will allow the text to wrap around the object.

 Warping text

  1. Click the 'Warp' section of the Text tab in the Context pane to see a variety of warping options, including Wave, Ribbon, Arch, and Arc.

  2. Select the text that you would like to warp and choose a warping style.

  3. Play with the Bend, Waves, Horizontal Distortion, and Vertical Distortion under each option to see how each affects the text. You can apply the warping to single lines of text or to entire paragraphs.

  4. Choose either Horizontal warping or Vertical warping for additional effects.

Changing tab settings

 

  1. From the Text tab in the Context pane, open the 'Tabs & Indents' section and select the text you'd like to edit.

  2. Change the first line, left indents, and right indents for the selected text with the arrows or by inputting a value and pressing the enter key on your keyboard.
  3. Modify the default tab settings with the arrows or by inputting a value. To add custom tab stops based on the default tab settings, use the + button.
