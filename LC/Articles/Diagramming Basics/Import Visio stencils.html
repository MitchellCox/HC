You can gain the benefits of Lucidchart without losing your shapes from Visio. Import your Visio stencils using the instructions below.

  1. Open the shape manager from inside the editor (using the “More Shapes” button or [M] on your keyboard).
  2. Click “Visio Stencil” button in the top-left corner of the window.
  3. Click “Choose File,” and select the Visio stencil file from your computer.
  4. Click “Upload.” Wait for the file to import (files larger than 1 MB can take longer than one minute). Imported stencils will appear in their own custom library.

Troubleshoot Visio imports

Visio import for Lucidchart is still in beta, so occasionally uploads will fail or diagrams will leave out a line or box here or there. Check out these suggestions to ensure that your upload is successful:

File size

If the import process takes longer than a few minutes, your file might be too large (>40 MB) and you’ll need to modify it. You can try:
  • Compressing user images: Typically, files are too large due to large user images placed inside the diagram. You can resolve this issue by optimizing the images to 72 dpi and ~1,000 pixels on the long side.
  • Dividing the document: If your file has multiple pages, try re-saving it to multiple files that have fewer pages and/or objects.
  • Saving as .vsd: VSD files are generally much smaller than VDX files so they should upload easier.

Version/format

You may get an error because you have uploaded a file in a format or version that we do not support. We currently support VSD files made with Microsoft Visio 2007 or later and VDX files made with Visio 2003 or later. If you have an older Visio file, consider re-saving it with a more recent version of Visio.

Because Lucidchart has supported VDX import for longer than VSD, you can also try saving it as a VDX for a higher fidelity result.
