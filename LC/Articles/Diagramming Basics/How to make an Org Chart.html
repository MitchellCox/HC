Create impressive organizational charts with Lucidchart's org chart software. Users can auto-create org charts or make one from scratch. This tutorial will walk you through the steps to make a great looking org chart.

Create new org chart
  • Enable the org chart shape library. For help, please view the tutorial on accessing more shape libraries here.
  • Drag out desired org chart shape from the toolbox, OR create from a template. For template help, see tutorial here.
  • Customize data as desired, OR import data (see process below).
Import data

Before importing your data, it is essential to make sure your data is formatted correctly. Your imported spreadsheet must include the following two columns to build the org structure:

  • Name: the full name of each individual must be listed in this column.
  • Reports To: the full name of the individual to whom each employee reports must be listed in a column. An individual with no formal “reports to” relationships (like a CEO) should not have any content in the corresponding “Reports To” field. This individual will be the head of the org chart.

Important Note: Make sure that the names in the “Name” column and “Reports To” column are the same. For example, if you have an employee “Margaret Thatcher,” her subordinates should not be reporting to “Thatcher, Margaret.” Additionally, every name listed in the "Reports To" column also needs to be listed in the "Name" column.

The “Name” and “Reports To” columns are the only necessary columns. Additional columns may be added and arranged as desired.

Your spreadsheet is now ready to be imported.

  1. From the Org Chart shape library in the left toolbox, select “Import Data.”

  2. From here, there are two ways data can be imported, by uploading a .csv file or by copying and pasting the information from a spreadsheet into the text field.
    • Uploading a .csv file: Click “Choose File” and select the desired file. Be sure that your .csv file looks like the image below. There should be no spaces and each column should be separated by commas.

    • Copy and paste data: Copy your spreadsheet data and paste it into the empty field.

  3. After you have either uploaded a .csv or copy and pasted your data, click 'Go'. Check the 'Use top rows as headers' box. This will make the column titles from your spreadsheet the headers in your org chart.

  4. Click 'Generate Org Chart' and the org chart will appear on the page.

Troubleshooting Tip: If you are missing connections between boxes, there is likely an error in your data. Make sure that the names in the “Name” column and “Reports To” column are the same. For example, if you have an employee “Margaret Thatcher,” her subordinates should not be reporting to “Thatcher, Margaret.” Additionally, every name listed in the "Reports To" column also needs to be listed in the "Name" column.

Add new employees

There are two ways to add new employees

In the context panel

  1. Open the context panel by selecting any box and clicking on the gear icon that appears next to it.
  2. In the bulleted list, press Enter to add a new employee.
  3. Press Tab to change an employee’s level in the organization.

On the canvas

  1. Draw a line extending from an existing box. A new box will appear and will “report to” the first box.
  2. To make the chart format consistent, open the context panel and select “Clean up layout.”
Edit data and chart structure

To edit personal information (name, role, phone number, etc.)

  1. Double click on the desired field
  2. Make desired changes

To add or remove fields

  1. Open the context panel (click on the gear icon next to a box).
  2. Select standardized fields (including the image field) by checking the selection boxes
  3. Add a customized field by selecting “+ Field”
  4. Remove a customized field using the garbage can icon

To add a photo

  1. Open the context panel (click on the gear icon next to a box)
  2. Select the image field. A gray image placeholder should appear in all org chart boxes.
  3. To choose a photo, copy and paste an image URL or use a picture uploaded to your image library.
  4. Add additional image URLs for other employees or click on an image place holder to select a picture from your image library.

To delete a photo

  1. Open the context panel (click on the gear icon next to a box)
  2. Click on the garbage can icon in the image field

Re-Standardize Layout

  1. Open the context panel (click on the gear icon next to a box)
  2. Select “Clean up layout”

To Change Position of Employee

  1. Open the context panel (click on the gear icon next to a box)
  2. In the bulleted list, hover over the space to the right of an employee. A double headed arrow should appear.
  3. Click on the arrow and drag the employee to the desired location
  4. Use Tab and Shift + Tab to edit employee level
Modify layout
  1. Choose a different layout by selecting from the format options in the context panel.

  2. Move an object manually by clicking and dragging.
  3. Select “Clean up layout” to undo manual movements.
Other tips and tricks

When manually moving an element, you may move secondary elements freely, but moving a primary element will move all of its associated elements, too.

You can edit org chart boxes just as you can edit other shapes—change their background color, line width, text options, etc. For help formatting shapes, see the tutorial here; for help formatting text, see the tutorial here.
