Lucidchart allows easy import of Visio, Gliffy, OmniGraffle, and AWS files. You can also export your diagrams as PDF, PNG, JPEG, VDX, or SVG files. See below for more information on importing and exporting. 

Diagram Import

It can be helpful to import diagrams from other tools or platforms. Lucidchart allows you to import diagrams from MS Visio (.vdx), Gliffy (.gxml, .gliffy), and Omnigraffle (.graffle, .graffle.zip).

Note: Diagram import is a premium feature offered on Pro, Team, and Enterpries accounts. To upgrade your account, check out our pricing page.

To import your diagrams, follow these steps:

  1. On the “My Documents” page, click the “Import” dialogue box and select the diagram type you wish to upload.

  2. Select the file you wish to upload and click “Import.”

  3. Click the diagram link to open the imported diagram.

Diagram Export

It can be helpful to export diagrams to other file formats. Lucidchart allows you to export your Lucidchart diagrams as .pdf, .png, .jpeg, .vdx, and .svg file formats.

Note: MS Visio export is a premium feature offered on Pro, Team, and Enterpries accounts. To upgrade your account, check out our pricing page.

To export your diagrams, follow these steps:

  1. Open the diagram you want to export.
  2. Open the “File” menu and select “Download As.”

  3. Select the file type you would like to export the diagram to, as well as the content you would like to export. If you are exporting the diagram as an image file, select the image quality of the export.

  4. Click “Download.”
