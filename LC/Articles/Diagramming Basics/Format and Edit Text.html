Lucidchart allows you to easily customize how text looks by formatting size, alignment, color, font, styling, and bulleting.

Text Size

To change text size, follow these steps:

  1. Select the text by double-clicking on it.

  2. Adjust the text size in the properties bar.

Alignment

To align text, follow these steps:

  1. Select the object by single-clicking on the object.

  2. Set the vertical and horizontal alignment by selecting the buttons on the properties bar.

Text Color

To change text color, follow these steps:

  1. Select the text by double-clicking it.

  2. Click on the text color box on the properties bar to open the color picker.

  3. Select the color you want and it will recolor the text.

Note: You can also manually enter a color’s hexidecimal code (#), modify the transparency (A), and access other advanced settings from the color drop-down menu.

Font

To change font, follow these steps:

  1. Select the text by double-clicking on it.

  2. Click the font and select the new font from the drop-down menu.

Note: You can also see the Add and Upload Fonts community post for more information. 

Styling

To add text styling, follow these steps:

  1. Select the text by double-clicking on it.

  2. On the properties menu, toggle Bold, Italic, Underline, Strike-through, Superscript, and Subscript.

  3. You can also use hotkeys to quickly apply styles to your text. You can see a full list of hotkeys by clicking on "Help."

Bulleting

To add bullets to text, follow these steps:

  1. Select text by double-clicking on it.

  2. Click the bulleting menu on the properties bar.

  3. Select from the list of bulleting styles.

  4. Add a new line of text by typing ‘Ctrl+Enter’.

Note: To end a bulleted list and continue typing, press ‘Shift+Enter’ to start a new line of text without a bullet point.
