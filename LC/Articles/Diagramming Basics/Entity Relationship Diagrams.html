Create Entity Relationship Diagrams (ERDs) with custom text, styling, and relationships between fields. Use the import feature to automatically generate database tables and schemas, ready to be placed on the canvas.

Automatically Create an ERD

The automated ERD feature enables you to automatically generate all of your database tables and schemas directly into the toolbox. When you drag related tables to the canvas, Lucidchart draws the relationship lines automatically. You can import updated database files at any time, and the fields will be updated in the toolbox and on the canvas.

Follow the steps below for automated ERD creation:

  1. Open the Shape Library Manager by clicking “More Shapes” in the bottom-left corner of the editor. Make sure the Entity Relationship library is turned on.

  2. Click the “Import” button inside the toolbox. An import window will appear.

  3. Select the Database Management System (DBMS) you use.
  4. Copy the provided query and paste it into your DBMS. Run the query.
  5. Export the results from your DBMS as a .tsv or .csv and upload to this page or copy the result and paste it into the open text area.

  6. Lucidchart then immediately imports and converts the text into an organized list of all databases, tables, and schemas. Each collapsible section represents a database, and the names inside correspond with the database tables. An additional subsection would indicate a schema.

Manually Create an ERD

Follow these steps to set up your ERD manually:

  1. Open the Shape Library Manager, and make sure the Entity Relationship library is turned on.
  2. Drag the desired entity object onto the canvas.
  3. Double-click on the text to enter text edit mode. Once you have selected one text box, you can easily move between text boxes by pressing Tab on your keyboard to go forward and Shift+Tab to go back. Press Enter or click off the object when finished.
  4. To change the size of the object, simply click and drag from one of the corners and extend to either side. The columns will auto-adjust as you move them to fit the content inside.



Add or remove fields

Use the context panel that appears to the right of each table to change the number of fields.

Modify arrow types
  1. Select the line with the arrow points you want to modify.
  2. From the properties bar, the drop-down menu allows you to select the style of ending point for each line. The default for an Entity Relationship arrow is as pictured below.

Manage fields
  1. If you want to reorganize the fields or change the text on your ERD, click “Manage Fields” in the context panel. From here, you can edit all text boxes directly.
  2. Click and drag the arrows on the left to move fields above and below each other.
  3. Click the + or - buttons on the right to add a field below the current one or to remove the field.
  4. Press “OK” to save changes.

Add style to ERD

You can easily style entity objects using the properties bar or the Graphic panel while the object is selected.

  • Modify text font, size, color, and formatting on the left side of the properties bar.
  • Modify line thickness and color in the center of the properties bar or in the Graphic panel.
  • Select the fill color in the properties bar or in the Graphic panel. The fill can include a solid color, gradient, or even an image.
  • Apply or alter a shadow in the Graphic panel.

Create relationships between objects

You can create a relationship line by clicking and dragging from any dot on the object. You can drag the line to another shape and create a relationship to a specific field OR release the cursor in the empty canvas to auto-prompt adding a new Entity object.

If you decide to move fields after you have created a relationship, the lines will move with the field. There is no need for reconnecting.



Export to SQL

With Lucidchart, you have the option to export your ERD to the DBMS of your choice. You can export single tables, or you can export the entire page—with the exception of connections between tables.

To export a single table:

  1. Click the gear icon on the top right of the shape, and then click “Export SQL.”
  2. A dialog will appear. Choose your DBMS from the drop-down menu on the right, and the text field below will change to the correct create statement.

To export your entire page:

  1. In the Entity Relationship shape library found in the left toolbox, click the “Export” button.
  2. A dialog will appear. Choose your DBMS from the drop-down menu on the right, and the text field below will change to the correct create statement for the entire page.
