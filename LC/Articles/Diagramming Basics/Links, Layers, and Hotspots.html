Make your diagram more interactive with links, layers, and hotspots. Currently, Free and Basic accounts do not have access to hotspots and layers. They can, however, add simple links to text and objects. Professional and Team accounts have access to hotspots and layers. Please see below for more information.

Creating Layers

A layer is an alternative view on a page. It can be shown or hidden, depending on the action assigned to the hotspot connected to it. Layers allow you to demonstrate user interaction before writing any code.

Layers are primarily accessed through the Layers panel, but we won’t discuss that tool here. If you are interested in learning more, read

Create a Layer

Click "Layers" from the Options Bar.

Choose the "+Layer" button.

Type a title for the layer. If you want to change the title, right-click on the current title and select “Rename”.

To add shapes to a layer, right-click on a shape and choose “Move to Layer”, or double-click on a layer and drag shapes onto the canvas.

Edit a Layer

From the Layers panel, double-click on a layer to edit it. The layer title will be highlighted, and you’ll also see a breadcrumb at the top of the editor that indicates which layer you’re in.

If you click on a layer on the canvas that you aren’t currently editing, you’ll see a bounding box around all objects on that layer (see image below). If you double-click that layer, you will enable editing abilities.

Hide/Show a Layer

To hide or view a layer, click its eye icon. Remember that if you use the quick preview hotkey (Ctrl+Shift or Cmd+Shift) and click on a hotspot, the eye will be temporarily unsynced.

Lock a Layer

Locking a layer will freeze all objects on the layer. You'll no longer be able to edit the layer. To lock multiple layers at a time, hold Ctrl (Cmd on Mac) and click each layer. The layer must be unselected in order to lock it.

Duplicate a Layer

Duplicating a layer will create a copy in the same place as the original layer. You can choose the page on which a duplicate layer will appear.

To duplicate a layer, right-click or choose the pen icon and select “Duplicate”. Then select the destination page or create a new one.

Reorder Layers

Click and drag the layer titles in the option menu to reorder layers.

Move to Layers

You can select objects from the document and move them to a specific layer of your choice. To do this, right-click and select 'Move to Layer'.

Choose the destination layer or create a new layer, then click 'OK'.

Copy to Layers

To copy objects to a layer of your choice, select the object(s) from the page or the layer you're currently editing.

Right-click and select “Copy to Layer”.

Choose the destination layer or create a new layer.

Sync Visibility

This feature establishes the default view for users who access a published webpage version of the diagram.

Click the eye icon of each layer until the page looks the way it should appear when it's published.

Click the Sync Visibility icon. When the icon is grayed out, syncing is complete. When you update your diagram, the webpage will also be updated.

Creating Links

Links can connect to another page within the document, an external page, or an email prompt. Follow these steps to create a simple link:

  1. Highlight the text or select the object you want to act as a link.
  2. Click on the link icon in the Properties bar.
  3. A window will appear with the option to select a link action. Choose to link to another page in your document, link to an external page, or prompt an email. Follow the prompt and add the URL you want to link to.
  4. Click “Done.” Note: Copy the URL directly from the address bar to ensure a proper connection.
  5. If you hover over the linked object or text, a tool tip appears, suggesting that you hold Shift+Ctrl (Shift+Cmd for Mac). You’ll notice that linked objects glow green. Click the object to initiate the action.
  6. Once a link is connected to an object, just right-click the object to change the link. You can also click the link icon again to edit it.<



Creating Hotspots

A hotspot is a designated area of the canvas that performs an action when selected in demo mode. Hotspots are ideal for UI mockups and demonstrations (premium libraries) and are available to Professional and Team accounts. They offer three simple linking actions (the same as the links in the Properties bar) and three premium actions that work with layers.

Follow these simple steps to create a hotspot and assign an action to it:

  1. Drag and drop the green hotspot shape from the standard shape library onto the selected object OR right-click on an object and select “Create Hotspot.”

  2. Click on the drop-down arrows next to “Choose an Action,” and select one of the six actions.

  3. Decide what you want the hotspot to do. Do you want it to show a layer all the time? Hide a layer like an “off” button? Toggle a layer like a light switch?

  4. If you’ve chosen a link option, enter the required information and click “Done.” If you’ve chosen an action that works with layers, you’ll need to select a layer on which the hotspot will act.

  5. If you have not created a layer yet…
    • Type a title for the layer into the “Create New Layer” field.
    • Click the [+] sign to create the layer.
    • Select the layer from the list, and click “Done.”

Note: You can select multiple layers for a single action by holding the Ctrl key (Command for Mac). To change an action, just double-click on the hotspot, which will open the action box.
