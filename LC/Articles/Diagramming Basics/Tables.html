Tables allow you to organize and visually display data. Paste content directly into cells and style them exactly how you want.

Add a table

Tables are so important that we gave them their own shape library! To add a table, use the following steps

  1. Open the shape manager (press “M” or click “More Shapes” in the lower left corner).
  2. Turn on the Tables shape library.
  3. Drag and drop a table onto the canvas.

Add content (including content from a spreadsheet)

Begin adding content by double-clicking on any cell. Press 'Tab' to advance to the next cell to the right.

Copy and paste from a spreadsheet

In addition to manually entering content, you can copy and paste from other sources.

Copy: Content that is tab-separated (like Google Spreadsheets) will paste with a preserved cell structure. If you copy a 10 x 10 cell selection in in Spreadsheets and paste it into a 3 x 3 table in Lucidchart, table size will automatically increase.

Selected cells from Google Spreadsheet 

Pasted cells in Lucidchart table from Google Spreadsheet

To paste data from Google Sheets into a Lucidchart table, select the first cell and press CTRL+V. Right clicking and selecting paste will not work. The table will automatically expand to the correct number of rows and columns to fit your data.

Keyboard and mouse controls

Tables introduce a few custom keyboard and mouse controls to make it easier to add and edit data inside the table. You can also add rows and columns from the context panel.

   

Select Cell: Select the table, then click on an individual cell. A dashed border around the cell indicates that it is selected.

Select Row/Column: Hover your mouse to the left edge of the table to select an entire row or hover along the top edge of the table to select an entire column.

Select Multiple Cells: Hold down Shift and click the first cell and last cell of a range to select multiple cells at a time. Hold down Ctrl (Command) to select multiple cells that are not next to each other.

Delete Cell Content: When cells are selected, you can simply press the Backspace or Delete keys and all content will be removed. You can also right-click on any cell and choose to clear an entire row or column. Note: This will also delete any text styling applied to cells.

Add/Delete a Row/Column: Change the number of rows or columns by selecting a cell and using the context panel buttons to insert and delete as needed. You can also right-click on the table and choose from the options in the pop-up menu.

Note: Increasing the count field for rows will add a new row and change the height of the table. Increasing the count field for columns will add a new column, but preserve the width of the table. Using the context panel buttons to insert a new column will change the width of the table.



Change Row/Column Width/Height: Hover over the border of a column or row and the cursor will change. This allows movement of borders in or out and up or down. Resize the entire table by clicking and dragging one of the corners of the table.

Note: As you increase the row size, the table size will increase. As you change the column size, the table size will stay the same. This means that other column sizes may change.

Add/delete columns and rows

Change the number of rows or columns by selecting a cell and using the context panel buttons to insert and delete as needed. You can also right-click on the table and choose from the options in the pop-up menu.

Note: Increasing the count field for rows will add a new row and change the height of the table. Increasing the count field for columns will add a new column, but preserve the width of the table. Using the context panel buttons to insert a new column will change the width of the table.



Merge cells

Select the cells you want to merge, then click the merge button inside the context panel. If you want to unmerge a cell, select the cell and click the second button.



Style and color

Lucidchart provides many ways to style your table. Here are a few options:

Text font and styling: Select a cell or multiple cells to change the font and make it bold, italic, or underlined. 

Banded rows and columns: In the context panel, find the “Banded” option and select either rows or columns to add color bands to your table and make the data easier to read.

Grid: In the context panel, select the grid option that best fits your data.

Cell color: Select a cell (or multiple cells), then use the top toolbar to modify the fill color and line color of the table.
