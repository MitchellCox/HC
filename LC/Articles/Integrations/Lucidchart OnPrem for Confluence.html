Lucidchart OnPrem User Manual

Important note: Lucidchart OnPrem for Confluence is a separate product from standard Lucidchart; it works only within Confluence Server, and does not integrate with existing Lucidchart accounts. Standard Lucidchart Team or Enterprise accounts do include free add-ons for an unlimited number of Confluence, JIRA, and HipChat instances and many more integrations.

If you have any questions about the differences between these two products, or would like help setting up your instance, contact our friendly support team at support@lucidchart.com

Installation

Lucidchart OnPrem installs automatically from the Atlassian Universal Plug-in Manager (UPM) or the manually via the Atlassian marketplace.

To Automatically Install from the Confluence UPM

  1. As an admin, open the Confluence administration menu and select "Add-ons"
  2. Select "Find new add-ons" in the left panel to open the Confluence UPM.
  3. earch the marketplace for "Lucidchart OnPrem" and click install to automatically install the add-on
  • Note that you may see a free add-on "Lucidchart for Confluence" with a circle icon. This add-on and others are free with a standard Team / Enterprise subscription to Lucidchart
  • Lucidchart OnPrem contains a server rack con (see below)

To Manually Install from the Marketplace

  1. Download the add-on from the Lucidchart OnPrem listing by selecting "Try It Free" or "Purchase Now"

  2. As an admin, open the Confluence administration menu and select "Add-ons"
  3. Select "Manage add-ons" in the left panel.
  4. Click "Upload add-on" and then select the file you downloaded in step #1.
  5. The add-on will be uploaded and automatically installed.

Installation Issues

For those using MySQL, there is one common error that may occur if your MySQL packet size setting is too small. This is common for larger Atlassian add-ons and is easily fixed using Atlassian's documentation below. https://confluence.atlassian.com/confkb/exceeds-max-allowed-packet-for-mysql-179443425.html

Please contact us if you experience any other issues with installation.

Using Lucidchart in Confluence

Insert Diagram at Top of Page

  1. Select Lucidchart Diagram from the Add drop-down menu from the top-right of your wiki page in Confluence.
  2. You will have the option to create a new Lucidchart document, insert an existing Lucidchart document, or directly import a Visio file. You can learn more about starting new documents and opening existing documents in our tutorials. 
  3. If you are importing a Visio file or creating a new diagram you will be taken directly into the Lucidchart editor. This short tutorial explains the basic components of the editor and how to get started.
  4. When you are finished editing your document in the Lucidchart editor, click the "Return to Confluence" button in the upper right corner to return to the wiki. Your wiki entry will be automatically added or updated.

Insert in Specific Page Location

  1. Insert a Lucidchart diagram into a specific section of your wiki-page using the macro tools inside the Confluence page editor.
  2. Click “Edit” at the top of the wiki page, then select  "Insert" and “Add Lucidchart Diagram.”
  3. Or, with the cursor inside the editor, type '{' to prompt the macro suggestion window. Start typing 'Lucidchart' and the Lucidchart macro will appear.
  4. When you select the Lucidchart macro you will be taken to the Lucidchart documents page where you have the option to begin a new Lucidchart document, insert an existing Lucidchart document, or directly import a Visio file. You can learn more about starting new documents and opening existing documents in our tutorials.

Make Changes to the Diagram Before Inserting

  1. If you decide you want to make changes to the diagram before inserting, click “Edit,” and you’ll be redirected to the Lucidchart Editor.
  2. When you’re finished editing, click “Return to Confluence” in the top right corner.
The LucidChart OnPrem Editor

The Lucidchart OnPrem editor is largely the same as our standard editor. This short tutorial explains the basic components of the editor and how to get started.

Some functionality available in the standard version of Lucidchart is not available in Lucidchart OnPrem. You may notice these missing features if you're familiar with our standard editor, but the powerful editor largely remains the same.

Visio Diagram and Stencil Import

Lucidchart OnPrem supports import of both Visio files and Stencil. It also allows the export of Visio diagrams. 

Visio File Import

  1. Go to edit a Confluence page and select "+ Insert" then add "Add Lucidchart Onprem diagram"
  2. In the left-side panel of the document picker, select "Import Visio / Gliffy File"

  3. Upload the Visio file and then select "Import." it may take a few moments to import the Visio diagram, which will then open up in Lucidchart.

Visio File Export

  1. Open the diagram within the Lucidchart editor
  2. Go to "File" >>> "Download As"
  3. Select the file type to download; options include Visio (VDX) format

Visio Stencil Import

  1. Open the diagram within the Lucidchart editor
  2. Select "More Shapes" in the bottom left corner of the editor to open the stencil / shape library manager

  3. Click the blue "Visio Stencil" button in the top left to open the import dialogue

  4. Choose the Visio stencil file and select upload to import
  5. The Visio stencil will then upload into a custom library within the shape manager

Note that some Visio stencils contain EMF images that do not work within Lucidchart OnPrem. These can be imported with web-based Lucidchart and then you may copy and paste these into a custom shape library within Lucidchart OnPrem.

Revision History

Every time you make a change to a Lucidchart diagram, a version is automatically saved in Confluence to prevent data loss. These are "minor" saves. By manually saving a diagram to clicking "Return to Confluence", you will automatically create a "major" version of the diagram and all previous minor changes will be erased to optimize performance.

If you would like to display (pin) or restore an older, major version of your Lucidchart diagram, rather than the most recently updated version, follow these steps:

  1. Start editing a Confluence page, select a Lucidchart diagram, and click Version in the pop-up panel.
  2. A dialogue with a list of previous versions will pop up.

  3. From here, you can choose to display or restore an older version.

To Display - to display an older version on the Confluence page, just click on the version and select "Save." The Confluence page will then show this version even if someone makes new changes to it. You can, of course, go back to displaying the latest version any time.

To help avoid possible confusion from other collaborators, the viewer will display an icon on the left side of the bottom toolbar to indicate that it's set to display an older version.

To Restore - restoring an older version makes it the latest version of the diagram and all subsequent edits will be made to the new, restored version. To do this, click the blue "Restore" button.
