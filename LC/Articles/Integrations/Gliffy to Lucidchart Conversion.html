Giffly files can be converted into a Lucidchart file within Confluence. To do this, follow these steps according to your instance type.

Confluence Cloud/Serve
  1. Click the "Attach" paper-clip symbol to access the Confluence page's attachments

  2. Select "Download All" to download the page's attachments

  3. From the pages editing menu, select "Lucidchart" from the "Insert" menu to add a Lucidchart diagram to the page

  4. Select "Import Gliffy" from the "Import" option within the Lucidchart macro

  5. Select the Gliffy file to upload

  6. Success! Click the blue document name to open the diagram with Lucidchart

  7. Make any necessary edits to the Lucidchart diagram. Once you have completed the editing process, click "Save and Close" to save the Lucidchart diagram

  8. Select the Lucidchart diagram from the dialogue box and click "Insert" to insert the diagram in your Confluence page

  9. Save your Confluence page

Confluence OnPrem

Lucidchart now allows users to automatically convert all Gliffy diagrams and macros on a page to Lucidchart.

  1. Go to a Confluence page with Gliffy diagrams
  2. Clicking (...) in the top right corner of the page. The menu should have "Convert Diagrams to Lucidchart." It willÂ NOT appear if any of the following apply:
    • There are no Gliffy diagrams on the page
    • The current user doesn't have edit permissions to the page
    • Lucidchart add-on has no license or expired license

  3. Clicking "Convert Gliffy diagrams to Lucidchart" takes you to a "Converting..." screen and back to the page after the conversion is done.
  4. Notes on conversion
    • Only Gliffy diagrams that are actually attached to the current page and are inserted into it (referenced by Gliffy macros) will be converted.
    • If a Gliffy macro points to a particular version, then that version will be converted.
    • First time the viewer loads the converted diagrams, it will have no loading thumbnail, but will have the size size as the original Gliffy diagram. After the viewer has loaded, however, it will make a thumbnail and save it in a new version of the attachment.
    • Gif images from Gliffy diagrams are not converted at the moment.

As this feature is currently in beta, please email us at support@lucidchart.com with any feedback on the conversion or the quality of the conversion to Lucidchart.
